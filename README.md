# FreshBite Premium Food Delivery Prototype

A polished static website demonstrating a premium food ordering experience using Bootstrap 5, professional layout, and interactive cart functionality.

## Features
- Modern Bootstrap-based responsive UI
- Hero section with premium content and quick stats
- Feature cards highlighting service benefits
- Searchable menu with product cards and external imagery
- Offcanvas order cart with quantity and total calculation
- Testimonials, newsletter, and contact sections
- Login and signup pages with Bootstrap form design

## Files
- `index.html` — Main landing page
- `style.css` — Custom styling and Bootstrap overrides
- `script.js` — Cart, menu search, and newsletter interactions
- `login.html` — Premium login page
- `signUp.html` — Premium sign-up page
- `auth.js` — Simple form behavior and redirect handling

## Usage
Open `index.html` in a browser to preview the responsive site. The cart and search are interactive in the static demo.
